# Overview

This project was a way for me to get introduced into data cleaning and piplining. Additionally,I thought that there might be families desiring to move to Alberta urgently due to unforseen circumestances or just wanting to move to Alberta in an easier way. This prompted me to build a housing search and ranking system to give an idea of listing based on completeness, location, and affordability.
